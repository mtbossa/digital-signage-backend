### How to debug Laravel API inside PHPStorm:
* Install Sail normally
* Send `XDEBUG_SESSION=session_name` or `XDEBUG_SESSION_START=session_name` with any request
> Read: **https://xdebug.org/docs/step_debug#manual-init**
